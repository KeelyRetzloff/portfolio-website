Project Background

This project was a WordPress portfolio for a published Photographer and Writer using the clients' own personal branding.

Goals and Objectives

The client needed the website to be easy to use and to update the content periodically. They also needed an album system to organize their Photography into categories that made sense and showcased their individuality and creativity.

Creative strategy

The project consisted of settiing up WordPress and choosing a theme that supported large amounts of images and adding HTML components and content to the site and styling accordingly. A plugin was added to manage photo storage and management. The client worked from a Macbook Pro using Safari, so consideration was put into browser optimization as well as troubleshooting WordPress-related issues for the client.

Technologies Used

WordPress, HTML, CSS, MacOS
